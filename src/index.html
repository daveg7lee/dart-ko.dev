Dart는 모든 플랫폼에서 빠른 앱을 개발하기 위해 클라이언트에 최적화된 언어입니다

Dart는 무료이고 오픈소스입니다

개발
반복적으로 수정하세요: 핫 리로드를 사용하여 실행하고 있는 앱의 결과를 즉시 확인하세요
모든 플랫폼에서
빠릅니다
모바일, 데스크톱 및 백엔드용 ARM 및 x64 시스템 코드로 컴파일합니다. 또는 웹용 자바스크립트로 컴파일이 가능합니다.
UI에 최적화
  • Isolate 기반의 동시성 과 병행되는 이벤트 기반 코드를 포함하는 사용자 인터페이스를 위한 성숙하고 완전한 async-await를 가지고 있습니다
  • Sound null safety, 컬렉션 확장을 위한 spread 연산자, 각 플랫폼의 UI 커스터마이징을 위한 collection if를 가진 사용자 인터페이스 구축에 최적화된 프로그래밍 언어입니다
  • 친숙한 문법으로 학습하기 쉬운 프로그래밍 언어 입니다
생산적인 개발
  • 핫 리로드를 사용하여 실행 중인 앱에서 즉시 결과를 확인하며 소스 코드를 반복적으로 수정하세요
  • 강력한 configurable tooling과 풍부한 정적 분석이 가능한 유연한 타입 시스템을 사용하여 코드를 작성하세요
  • 코드에디터에서 profiling, logging, 및 debugging 하세요
모든 플랫폼에서 빠릅니다
  • 빠른 실행을 위해 AOT-compile을 사용하여 앱을 네이티브 머신 코드로 컴파일합니다
  • 완전하고 성숙하며 빠른 자바스크립트 컴파일러 로 웹을 처리하세요
  • 앱을 지원하는 백엔드 코드를 단일 프로그래밍 언어로 작성하여 실행하세요
